# annoying_alarm

## Getting Started

kalo mau pake image di programnya buat image jadi constant dulu di ```lib\core\utils\image_constant.dart``` terus pake CustomImageView() buat nampilin

contoh :
```
CustomImageView(
  imagePath: ImageConstant.imgLogoLandingPage,
  height: 214.0,
  width: 252.0,
)
```
